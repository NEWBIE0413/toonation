# toonation
크롤링을 사용한 간단한 투네이션 후원감지 파이선 코드입니다.
* 감지시에 새로고침이 사용되지 않습니다.

# 사용법
투네이션 통합 알림창 URL을 복사하여 소스코드 내 주소에 붙혀넣기 합니다. (사진 참조)
